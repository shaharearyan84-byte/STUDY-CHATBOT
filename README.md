# 🎓 Student AI Assistant

## 📌 About the Project

Student AI Assistant is a web-based AI chatbot developed to help students with their academic questions and learning needs. The application provides a simple, interactive, and user-friendly interface where students can ask questions and receive instant responses.

This project was built using **Python**, **Flask**, **HTML**, **CSS**, and **JavaScript** in **Visual Studio Code**.

---

## ✨ Features

- 🤖 AI-powered chatbot
- 📚 Helps students with educational queries
- 💬 Interactive chat interface
- ⚡ Fast response system
- 🎨 Modern and attractive UI
- 📱 Responsive design
- 🌐 Web-based application

---

## 🛠️ Technologies Used

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- VS Code
